```python
Data = [
[1, 'PR생성', '9:00'],
[1, 'PR분석', '10:00'], 
[1, '견적요청_생성(담당자)', '10:30'], 
[1, '견적_분석요청', '11:00'], 
[1, '견적_수정요청', '11:30'], 
[1, '견적_분석요청', '12:00'], 
[1, '견적_수정요청_(담당자)', '12:30'], 
[1, '견적_분석요청', '13:00'], 
[1, '견적_수정요청', '13:30'], 
[1, '견적_분석요청', '14:30'], 
[1, '견적_수정요청', '15:00'], 
[1, '견적_분석요청', '15:30'], 
[1, '견적서_송부', '16:00'], 
[1, 'PO생성', '16:30'], 
]

'''
반복된 프로세스들을 한눈에 확인하고 관리하기 위하여 관련된 칼럼들을 새로 추가해준다.

1. 변수 ‘analyze_count’를 정수형 변수로 만든다.
2. 변수 ‘amend_count’를 정수형 변수로 만든다.
3. 레코드를 한줄씩 불러온다
4. 만약 두번째 열이 견적_분석요청이면 변수 analyze_count를 증가(정확히 일치)
5. 만약 두번째 열 안에 견적_수정요청이 있으면 amend_count증가(포함)
6. 위의 두 조건을 만족하지 않으면 아무것도 하지x
'''

analyze_count = 0
amend_count = 0
new_list = []

for d in Data:
    if d[1] =='견적_분석요청':
        analyze_count+=1
        d.extend([analyze_count, 'None'])
    elif '견적_수정요청' in d[1]:
        amend_count+=1
        d.extend(['None', analyze_count])
    else:
        d.extend(['None', 'None'])
    new_list.append(d)   # new_list = new_list.append(d) 하지 않도록 주의


print(new_list)
```

#### append와 extend
a = [1,2,3]

리스트 형태 그대로 추가하려면 append() 
- a.append([4,5]) --> a = [1, 2, 3, [4, 5]]

요소들만 기존의 리스트에 추가하고 싶다면 extend()
- a.extend([4,5]) --> a = [1, 2, 3, 4, 5]
